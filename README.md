# Tutorial Linux (pt-br)
Repositório de tutoriais para instalação em sistemas operacionais linux. O arquivo `.tex` também é disponibilizado, você pode utilizar um compilador `.tex` para compilar o arquivo para pdf. 

1. Artigo Debian Mínimo - Tutorial de Instalação de Sistema Operacional Debian-Linux customizada a partir de instalação mínima com servidor gráfico X11, compositor picom e gerenciador de janelas em blocos i3. O arquivo `DEB_install.sh` serve para instalar rapidamente o sistema Debian com (i3+picom), edite e customize o arquivo antes de utilizar.
