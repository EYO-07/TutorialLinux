# Linux Box 

Installer Scripts, Tutorials, Workflows, Config Files for easy linux minimal setups.

1. template_x_resources : `cp ./template_x_resources ~/.Xresources` for basic terminal settings in x11.
2. template_bash_aliases : `cp ./template_bash_aliases ~/.bash_aliases` for some verbose commands in terminal.
3. template_inputrc : `cp ./template_inputrc ~/.inputrc` for customized terminal input behaviour.
